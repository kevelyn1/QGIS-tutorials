# QGIS-tutorials
QGIS-tutorials for mostly beginners level to create maps in QGIS

## Download QGIS
To install QGIS, please go to webpage: https://qgis.org/en/site/forusers/download.html
Depending on your operation system, please download and install:  QGIS Version 3.10 

## Download data
Global level free vector data for administrative borders, waterbodies and elevation from Natural Earth:  
https://www.naturalearthdata.com/

Over 500 sites providing freely available geographic datasets:
http://freegisdata.rtwilson.com/
